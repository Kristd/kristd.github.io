golang的int转换为duration类型

2017/08/07

Categories: Dev Tags: Golang

问题

在调用time.sleep的时候如果传入一个整形的变量会有如下报错：

var t int
time.Sleep(time.Second * t)  // error 

>> invalid operation: time.Second * t (mismatched types time.Duration and int)`

解决办法

golang的time.Duration类型背后是int64

type Duration int64`

我们需要对整形t做一个类型转换才能正确调用该函数

time.Sleep(time.Second * time.Duration(t))  // success
