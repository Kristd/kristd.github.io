Golang使用断言进行类型转换

2017/06/28

Categories: Dev Tags: Golang

在对interface做类型转换时需要使用断言来转换成正确的类型。

测试代码

type R struct {
	Config []map[string]interface{}
}

r := &R{}

//转换成字符串
key1, ret := r.Config[0]["key1"].(string)
if ret {
    fmt.Println("success")
}

//转换成数组
array1, ret := r.Config[0]["array1"].([]interface{})
if ret {
    fmt.Println("success")
}
